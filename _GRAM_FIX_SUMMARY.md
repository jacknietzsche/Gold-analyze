### P1：评分体系不完整

**问题1**: 缺少"经济扩张"因子
- WGC GRAM框架包含4个因子：经济扩张、风险与不确定性、机会成本、趋势动能
- 当前实现只有3个因子：机会成本、风险与不确定性、供需格局（不是"经济扩张"）

**修复方案**:
1. 添加"经济扩张"因子（GDP增长率、工业增加值、消费者信心指数）
2. 将"供需格局"改名为"供需格局"，并扩展数据来源（央行购金 + ETF持仓 + 珠宝需求 + 工业需求）

---

**问题2**: 权重分配可能主观
- 当前权重：机会成本40%、风险与不确定性25%、供需格局20%、趋势动能15%
- 未基于历史数据回测优化

**修复方案**:
1. 使用历史数据回测，计算各因子与黄金收益率的IC（信息系数）
2. 基于ICIR（IC稳定性）分配权重
3. 参考WGC官方文档的权重建议

---

**问题3**: 评分阈值可能主观
- 当前阈值：国债收益率<2.0%=8分，VIX>30=9分等
- 未基于历史数据分位数确定

**修复方案**:
1. 使用历史数据，分析各因子与黄金收益率的相关性
2. 根据相关性确定评分阈值（如历史分位数）
3. 使用滚动窗口优化阈值

## 📊 评分公式检查

### 机会成本因子
**当前公式**:
```python
oc_avg = (rate_score + dxy_score) / 2.0
```

**问题**: 简单平均，未考虑利率和美元对黄金的影响程度不同

**改进方案**:
```python
# 基于历史回归的权重
rate_weight = 0.6  # 利率对黄金影响更大
dxy_weight = 0.4
oc_avg = rate_score * rate_weight + dxy_score * dxy_weight
```

---

### 趋势动能因子
**当前公式**:
```python
momentum = (r1d + r5d + r20d) / 3.0
```

**问题**: 简单平均，未考虑时间衰减（近期收益应权重更高）

**改进方案**:
```python
# 指数加权移动平均
weights = [0.5, 0.3, 0.2]  # 1日, 5日, 20日
momentum = r1d * weights[0] + r5d * weights[1] + r20d * weights[2]
```
