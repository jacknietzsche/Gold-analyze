# 黄金量化综合策略框架文档

## 1. 策略概述

本策略框架融合了传统量化策略与现代技术分析方法，形成一个完整的黄金交易系统。通过综合多维度因子分析、技术指标验证和传统量化模型，实现了策略的自动化执行与动态调整。

### 1.1 策略目标
- 实现黄金价格的精准预测和交易信号生成
- 结合基本面、技术面和市场情绪多维度分析
- 提供严格的风险控制机制
- 实现策略的自动化执行和动态调整

### 1.2 适用场景
- 黄金期货(GC=F)日度/周度交易
- 黄金ETF(GLD)投资决策
- 投资组合中的黄金配置调整
- 宏观经济环境变化时的黄金策略调整

## 2. 策略架构

### 2.1 核心模块

| 模块 | 功能 | 描述 |
|------|------|------|
| 数据获取模块 | 数据采集与处理 | 获取黄金价格、美元指数、实际利率等因子数据 |
| 因子打分体系 | 多因子综合分析 | 7个核心因子的信号生成与综合打分 |
| 技术分析模块 | 技术指标计算 | 均线、RSI、MACD、布林带等技术指标计算 |
| 传统量化策略 | 经典策略实现 | 线性回归、多因子打分、事件驱动、规则化策略 |
| 信号生成模块 | 综合信号判断 | 多策略信号加权整合 |
| 交易执行模块 | 自动交易执行 | 基于信号的交易操作 |
| 风险控制模块 | 止损与仓位管理 | 止损设置与风险控制 |

### 2.2 数据流程图

```
[数据获取] → [因子计算] → [技术分析] → [传统策略执行] → [信号整合] → [交易执行] → [风险控制]
```

## 3. 因子打分体系

### 3.1 核心因子

| 因子 | 信号规则 | 数据频率 | 权重 |
|------|---------|---------|------|
| **黄金价格** | 突破前高（20日高点）→ +1；跌破前低（20日低点）→ -1 | 日度 | 1 |
| **美元指数 (DXY)** | 连续2日下跌 → +1；连续2日上涨 → -1 | 日度 | 1 |
| **实际利率（10年期TIPS）** | 连续3日累计下降>10bp → +1；连续3日累计上升>10bp → -1 | 日度 | 1 |
| **黄金ETF成交量（GLD）** | 连续2日放量（成交量>20日平均1.5倍）上涨 → +1；连续2日放量下跌 → -1 | 日度 | 1 |
| **VIX恐慌指数** | 单日涨>10% → +1；单日跌>10% → -1 | 日度 | 1 |
| **黄金/标普500比值** | 连续3日上升 → +1；连续3日下降 → -1 | 日度 | 1 |
| **CFTC投机净多头** | 低于20%分位（过去1年）→ +1；高于80%分位（过去1年）→ -1 | 每周 | 1 |

### 3.2 综合得分计算
- 累加以上7个因子的得分（周度因子CFTC在每周五更新后纳入计算，其余时间沿用前值）
- 信号判定标准：
  - ≥+2 → 生成明确看多信号
  - ≤-2 → 生成明确看空信号
  - -1~+1 → 维持震荡/观望状态，不生成交易信号

## 4. 技术分析模块

### 4.1 技术指标

| 指标 | 参数 | 信号规则 |
|------|------|---------|
| **均线系统** | 20日SMA | 价格收于均线上方 → 多头趋势；反之 → 空头趋势 |
| **RSI指标** | 14周期 | RSI<30 → 超卖；RSI>70 → 超买 |
| **MACD指标** | (12,26,9) | 金叉 → 买入信号；死叉 → 卖出信号 |
| **布林带** | (20,2) | 价格触及下轨且RSI<35 → 潜在买入；价格触及上轨且RSI>65 → 潜在卖出 |
| **支撑/阻力位** | 前20日高低点 | 价格回踩支撑位 → 买入机会；价格触及阻力位 → 卖出机会 |
| **价格形态** | 多种形态 | 头肩底 → 买入形态；头肩顶 → 卖出形态 |

## 5. 传统量化策略

### 5.1 线性回归择时模型
- **原理**：基于多因子线性回归预测黄金未来收益率
- **因子选择**：通过VIF检验筛选低相关性因子
- **信号生成**：预测收益率>1% → 买入；<-0.5% → 卖出

### 5.2 多因子打分模型
- **原理**：基于因子权重加权计算综合得分
- **因子权重**：宏观因子(60%) > 供需因子(15%) > 情绪因子(15%) > 技术因子(10%)
- **信号生成**：得分处于历史80%分位以上 → 买入；20%分位以下 → 卖出

### 5.3 事件驱动策略
- **原理**：基于特定事件的发生生成交易信号
- **事件类型**：通胀超预期、实际利率变化、VIX恐慌、季节性、ETF资金流、技术指标信号
- **信号生成**：事件信号强度加权平均，>0.3 → 买入；<-0.3 → 卖出

### 5.4 规则化趋势/反转策略
- **原理**：基于技术指标和基本面因子的组合规则
- **规则示例**：
  - 多头排列 + 实际利率下行 → 买入
  - RSI超卖 + ETF净流入 → 买入
  - 强动量 + 低波动率 → 买入
  - 布林带超卖 + 美元疲软 → 买入
- **信号生成**：买入规则数 > 卖出规则数 → 买入；反之 → 卖出

## 6. 综合信号生成

### 6.1 信号加权整合
- **因子信号**：权重 1.0
- **技术信号**：权重 0.8
- **线性回归信号**：权重 1.2
- **多因子打分信号**：权重 1.0
- **事件驱动信号**：权重 0.8
- **规则化策略信号**：权重 0.8

### 6.2 最终信号判定
- 加权平均信号 > 0.3 → 看多信号
- 加权平均信号 < -0.3 → 看空信号
- 其他 → 中性信号

## 7. 交易执行流程

### 7.1 入场规则
- 在确认共振信号后，等待价格回踩关键支撑位（看多信号）或阻力位（看空信号）时执行入场操作
- 多头入场：价格回踩支撑位附近
- 空头入场：价格触及阻力位附近

### 7.2 风险控制
- **止损设置**：
  - 多头头寸：关键支撑位下方0.5~1%处
  - 空头头寸：关键阻力位上方0.5~1%处
- **仓位管理**：单笔交易风险敞口不超过总资金的2%

### 7.3 离场规则
- 每日收盘后复查因子综合得分，当得分转为中性区间（-1~+1）时执行离场
- 当价格突破止损位或技术指标出现反向信号时立即离场
- 每周日对策略进行全面复盘，评估因子有效性和策略表现

## 8. 策略回测与优化

### 8.1 回测指标
- **年化收益率**：策略年度化回报率
- **最大回撤**：策略历史最大亏损幅度
- **夏普比率**：风险调整后收益
- **胜率**：盈利交易占比
- **盈亏比**：平均盈利/平均亏损

### 8.2 参数优化
- **因子权重优化**：基于历史数据回测，调整各因子权重
- **技术指标参数**：优化均线周期、RSI阈值等参数
- **止损比例**：根据市场波动性调整止损幅度
- **入场时机**：优化价格回踩确认条件

### 8.3 过拟合防范
- 使用交叉验证避免过拟合
- 定期回测检验策略有效性
- 监控策略表现，及时调整参数

## 10. 策略优势与风险

### 10.1 优势
- **多维度分析**：结合基本面、技术面和市场情绪
- **信号共振**：多策略信号确认，提高信号可靠性
- **风险控制**：严格的止损和仓位管理
- **自动化执行**：完整的交易执行流程
- **可扩展性**：模块化设计，易于扩展和修改

### 10.2 风险
- **数据质量风险**：数据源不稳定或数据延迟
- **模型风险**：模型假设与实际市场不符
- **市场风险**：极端市场条件下策略失效
- **执行风险**：交易执行延迟或滑点

### 10.3 风险缓解措施
- 多数据源备份，确保数据可靠性
- 定期模型评估和调整
- 压力测试，评估极端市场条件下的表现
- 优化交易执行，减少滑点影响
